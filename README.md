# TensileTesterApp - 拉力计数据处理与可视化工具

这是一个专为科研数据分析设计的 Python 应用程序，旨在高效处理拉力计生成的 CSV 原始数据，提供数据清洗、摩擦系数换算、平滑处理及可视化导出功能。

## 🌟 核心功能

- **数据预处理**：自动处理正负值（统一转换为绝对值），适配拉力与推力混合数据。
- **双轴数据过滤**：支持自定义 **受力范围 (N)** 和 **时间范围 (S)**，剔除不相关的实验阶段。
- **摩擦系数换算**：支持自定义摩擦系数，自动计算并生成 `Friction coefficient processing` 处理列。
- **高级平滑算法**：内置 Savitzky-Golay 滤波器，参数可调（窗口大小），实时预览平滑效果。
- **实时统计**：自动计算并展示过滤后原始数据的平均拉力值。
- **结果导出**：支持一键导出加工后的 CSV 数据表（包含原始与处理列）及高分辨率可视化图表（PNG/JPG）。

## 🚀 快速开始

### 1. 环境准备
确保您的计算机已安装 Python 3.8+。

### 2. 安装依赖
在终端中运行以下命令安装必要的库：
```powershell
pip install pandas customtkinter matplotlib scipy pillow
```

### 3. 运行应用
```powershell
python main.py
```

## 🛠️ 使用说明

1. **选择文件**：点击“选择数据文件”加载您的实验 CSV。
2. **设置参数**：
   - **数值过滤范围**：定义您关注的受力区间。
   - **时间过滤范围**：限制分析的时间段（默认 `all` 为全选）。
   - **摩擦系数**：输入您的实验摩擦系数进行数据折算。
   - **平滑窗口**：点击 `?` 图标查看设置建议。
3. **计算与查看**：点击“执行计算与可视化”，右侧将展示处理后的波形。
4. **保存结果**：满意后点击保存按钮导出数据或图表。

## 📦 打包指南

如果您希望生成独立的 `.exe` 文件：
```powershell
pip install pyinstaller
pyinstaller --noconsole --onefile --icon="app_icon.ico" main.py
```
